## FAQ

### General Questions

**Q: I forgot my password. How do I reset it?**
A: Click "Forgot Password" on the login page, enter your email, and follow the instructions sent to your email.

**Q: How do I change my profile picture?**
A: Go to Profile > Edit Profile > Upload Photo > Save

**Q: How do I enable notifications?**
A: Go to Settings > Notifications > Enable desired notifications > Save

### Admin Questions

**Q: How do I add multiple students at once?**
A: Go to Students > Import > Download template > Fill data > Upload CSV

**Q: How do I generate fee receipts in bulk?**
A: Go to Finance > Receipts > Select date range > Generate Bulk Receipts

**Q: How do I backup school data?**
A: Go to Settings > Data Management > Create Backup

### Teacher Questions

**Q: Can I edit attendance after submission?**
A: Yes, go to Attendance > History > Select date > Edit > Submit

**Q: How do I download my class list?**
A: Go to My Classes > Select class > Export > Download CSV

**Q: Can I send attachments in messages?**
A: Yes, click the attachment icon when composing a message

### Parent Questions

**Q: Can I pay fees for multiple children together?**
A: Yes, go to Fees > Select all pending fees > Pay All

**Q: How do I download previous report cards?**
A: Go to Academics > Report Cards > Select exam > Download

**Q: Can I track the school bus?**
A: Yes, go to Transport > Track Bus (if enabled by school)

### Student Questions

**Q: How do I submit assignments online?**
A: Go to Assignments > Select assignment > Upload file > Submit

**Q: Can I view my attendance history?**
A: Yes, go to Attendance > View History

**Q: How do I check library book availability?**
A: Go to Library > Search > Check availability status
