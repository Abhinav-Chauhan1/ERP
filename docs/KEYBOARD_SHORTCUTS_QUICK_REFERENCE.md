# Keyboard Shortcuts Quick Reference

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `/` | Focus Search | Quickly focus the global search input to find students, teachers, documents, etc. |
| `Ctrl+K` (or `Cmd+K`) | Command Palette | Open the command palette for quick navigation and actions |
| `?` | Help | Show all available keyboard shortcuts |
| `Esc` | Close | Close open dialogs, modals, and dropdowns |

## Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `↑` / `↓` | Navigate Lists | Move up and down through list items |
| `Enter` | Select | Select the currently highlighted item |
| `Tab` | Next Field | Move to the next form field or interactive element |
| `Shift+Tab` | Previous Field | Move to the previous form field or interactive element |

## Command Palette Actions

When you open the command palette with `Ctrl+K`, you can:

### Navigation Commands
- Go to Dashboard
- Go to Students
- Go to Teachers
- Go to Parents
- Go to Classes
- Go to Attendance
- Go to Finance
- Go to Reports
- Go to Library
- Go to Transport
- Go to Admissions
- Go to Certificates
- Go to Settings

### Quick Actions
- Add New Student
- Add New Teacher
- Mark Attendance
- Generate Report

## Tips

1. **Search First**: Press `/` to quickly search for anything in the system
2. **Command Palette**: Use `Ctrl+K` when you know what you want to do but don't want to navigate through menus
3. **List Navigation**: When viewing lists, use arrow keys to quickly navigate without reaching for the mouse
4. **Form Efficiency**: Use Tab to move through form fields quickly
5. **Help Anytime**: Press `?` if you forget a shortcut

## Browser Compatibility

All keyboard shortcuts work in:
- Chrome/Edge (Chromium)
- Firefox
- Safari
- Opera

## Accessibility

All keyboard shortcuts are designed to work seamlessly with:
- Screen readers
- Keyboard-only navigation
- High contrast modes
- Assistive technologies

## Mobile

Keyboard shortcuts are primarily for desktop users. On mobile devices:
- Touch interactions work normally
- No keyboard shortcuts interfere with mobile gestures
- All features remain accessible through the UI

## Need More Help?

Press `?` anywhere in the application to see the full keyboard shortcuts help modal with detailed information about all available shortcuts.
